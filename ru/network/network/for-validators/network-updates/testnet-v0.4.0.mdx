Это обновление сети приносит следующие изменения:

* Ребрендинг нативного токена \$BIT в \$MNT
* Добавляет функции доказательства мошенничества
* Включает делегирование для модулей доказательства мошенничества и TSS
* Открывает сеть Mantle DA для внешних сторон

## Инструкции для операторов узлов

Обновление тестовой сети планируется запустить на высоте блока \*\*11,000,000.\*\* Операторам узлов необходимо обновить программное обеспечение своих узлов до этой высоты блока, чтобы избежать непредвиденного поведения узла.

Вы можете обновить версию вашего узла до v0.4.0 с помощью Docker. Теги образов следующие:

* **Для узлов L2** - `l2geth:v0.4.0-1`
* **Для сервиса DTL** - `dtl:v0.4.0`
