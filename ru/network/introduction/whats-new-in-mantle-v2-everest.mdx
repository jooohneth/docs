После более чем года стабильной работы Mantle Tectonic, мы с гордостью представляем **Everest Upgrade**— значительный шаг вперед для сети Mantle. Это обновление не только укрепляет нашу существующую инфраструктуру, но и полностью готовит нас к предстоящему **Ethereum Pectra Upgrade**, обеспечивая бесперебойную совместимость с сетью Mantle и закладывая прочную основу для будущего развития. Одним из ключевых моментов обновления Everest является официальный запуск **EigenDA**, который представит революционное решение для доступности данных в сети, значительно повышая эффективность управления данными и безопасность.

## Улучшения в Mantle v2 Everest по сравнению с Mantle v2 Tectonic

### Интеграция EigenDA

Mantle поддерживает глубокое сотрудничество с EigenDA. До запуска основной сети EigenDA, Mantle использовал Mantle DA (сторонний слой DA, работающий на EigenDA) в качестве своего слоя DA. Поэтому, когда EigenDA запустился, мы немедленно завершили миграцию в тестовой сети и, после тестирования в производственной среде, окончательно мигрировали в основной сети Mantle. Для получения подробной информации о EigenDA, пожалуйста, ознакомьтесь [here](/network/system-information/off-chain-system/eigenda).

### Адаптация L2 для обновления Ethereum Pectra

Ethereum готовится к долгожданному обновлению Pectra. Как L2 Ethereum, Mantle немедленно начал работу по адаптации к изменениям, внесенным обновлением Pectra, включая новое `RequestsHash` field в заголовках блоков из EIP-7685, и новый `SetCodeTx` тип транзакции, введенный в EIP-7702.

### Введение новых функций

С момента создания на основе OP Stack Bedrock, Mantle был привержен улучшению функциональности op-geth через расширяемую разработку. В этом обновлении была введена поддержка нового предкомпилированного контракта RIP-7212, а также добавлен `eth_getBlockReceipts` интерфейс jsonRPC.

### Прекращение поддержки функции MetaTX

MetaTX, независимо разработанный Mantle, предоставлял функциональность спонсирования комиссии за газ, направленную на минимизацию барьеров для пользователей, входящих в экосистему Mantle. С введением EIP-7702 в обновлении Ethereum Pectra, Mantle будет предоставлять ту же функциональность спонсирования комиссии за газ на его основе, поэтому MetaTX больше не будет поддерживаться.[](https://docs-v2.mantle.xyz/intro/whats-new-in-mantle-v2#enhancements-in-mantle-v2-tectonic-compared-to-mantle-v1)
