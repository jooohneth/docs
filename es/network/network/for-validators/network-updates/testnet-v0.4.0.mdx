- title: Red de pruebas v0.4.0

Esta actualización de la red trae los siguientes cambios:

* Cambio de marca del token nativo \$BIT a \$MNT
* Agrega características de prueba de fraude
* Habilita la delegación para los módulos de prueba de fraude y TSS
* Abre la red Mantle DA para partes externas

Para más detalles sobre cómo se incluyeron las características mencionadas anteriormente en el código base, siga los enlaces a continuación para navegar al repositorio de GitHub de Mantle.

## Instrucciones para Operadores de Nodos

La actualización de la red de pruebas está programada para entrar en funcionamiento en la altura de bloque \*\*11,000,000.\*\* Los operadores de nodos deben actualizar el software de sus nodos antes de esa altura de bloque para evitar un comportamiento inesperado del nodo.

Puede actualizar la versión de su nodo a v0.4.0 usando Docker. Las etiquetas de imagen son las siguientes:

* **Para nodos L2** - `l2geth:v0.4.0-1`
* **Para el servicio DTL** - `dtl:v0.4.0`
